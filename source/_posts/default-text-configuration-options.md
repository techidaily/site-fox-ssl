## Default Strings Tab

Default strings can be used in subsequent projects, and their[localization identifiers](https://tools.techidaily.com/advancedinstaller/products/) are associated with Advanced Installer's default controls and elements. 

![Important](https://cdn.advancedinstaller.com/svg/common/IconMessageInfo.svg)There are two ways of translating default strings to a new dictionary and including it into your project. One involves selecting the untranslated language from the [Languages Tab](https://tools.techidaily.com/advancedinstaller/products/) and then defining each string in Advanced Installer's predefined dictionary editor described in this article. The other involves manually editing the dictionary file and is described in [Translating and using a new language](https://tools.techidaily.com/advancedinstaller/products/) tutorial.

![Default strings tab](https://cdn.advancedinstaller.com/img/ui/default-strings-tab.png "Default strings tab")

### Store changes to repository

Save changes to your repository. Each language will be stored in separate dictionaries.

### Store changes to a specific location

Save changes as a dictionary file to a specific location on your computer. All languages will be stored in a single dictionary.

### Inspect repository

Manage dictionaries with the [Repository Manager](https://tools.techidaily.com/advancedinstaller/products/).
