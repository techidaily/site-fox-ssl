### Product Code

In this section, you can set the package's Product Code. Each language uses a different Product Code, so you can change it for a single language or for all languages. When a Product Code is selected, you can use the \[Edit... \] button to modify it. Also, you can use the \[Generate All \] button to change all Product Codes.

![Caution!](https://cdn.advancedinstaller.com/svg/common/IconMessageWarning.svg)To preserve the language on maintenance and uninstall when using multilingual packages, make sure to use the same ProductCode for all languages.
