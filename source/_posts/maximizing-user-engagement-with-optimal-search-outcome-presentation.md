## Search Results Dialog

 Select in the searches list the desired search property and its value will be displayed in the bottom edit field.

![Search Results](https://cdn.advancedinstaller.com/img/dialog/search-results.png "Search Results")  

![Note](https://cdn.advancedinstaller.com/svg/common/IconMessageNote.svg)The string after the "1: " characters (the searches number) represents the actual value stored in the searches property.
